**Chirping Astro** is a single-template, Chirpy-inspired theme for Astro v6.

It is built with:

- **Astro 6.x** + Content Collections + MDX
- **Tailwind CSS v4** with the `@tailwindcss/vite` plugin
- **daisyUI v5** for accessible components and theming
- **Pagefind** for instant, low-cost search
- **Giscus** for GitHub Discussions–powered comments
- **EN + FR** internationalization (EN at the URL root)

The goal is to give you a delightful Chirpy-like reading experience while
keeping the code idiomatic and easy to extend.
