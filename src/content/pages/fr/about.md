**Chirping Astro** est un thème mono-template inspiré de Chirpy pour Astro v6.

Il est construit avec :

- **Astro 6.x** + Content Collections + MDX
- **Tailwind CSS v4** via le plugin `@tailwindcss/vite`
- **daisyUI v5** pour des composants accessibles et thématisables
- **Pagefind** pour une recherche instantanée et frugale
- **Giscus** pour des commentaires propulsés par GitHub Discussions
- Internationalisation **EN + FR** (l'anglais à la racine de l'URL)

Objectif : offrir une expérience de lecture proche de Chirpy tout en
gardant un code idiomatique et facile à étendre.
